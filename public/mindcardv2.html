使用说明

此心智模型卡片模板支持导入JSON数据批量生成卡片，并可通过左右按钮切换查看不同卡片。

数据格式要求：

请生成一个 JSON 数组，内容为认知偏差/思维谬误。每个 JSON 对象必须包含以下字段： 

        {
      "title": "偏差名称（中文）",
      "subtitle": "用一句简短话总结该偏差的核心特征",
      "icon": "图标名称（参考FontAwesome，如'brain'）",
      "description": "对该偏差的简要说明，要求简洁清晰，通俗易懂",
      "example": "一个真实或常见的生活/工作场景中的例子",
      "importance": 数值（0-100表示重要程度）,
      "tags": ["标签1", "标签2", ...]
  }

数据生成提示：

请生成一个 JSON 数组，内容为不少于 50 条常见的认知偏差/思维谬误。

标签数组至少包含 2 个分类，例如："决策偏差"、"信息偏差"、"社会心理"、"风险感知"、"记忆偏差"等。
